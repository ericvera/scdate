# Function: getTimestampNow()

> **getTimestampNow**(`timeZone`): [`STimestamp`](../classes/STimestamp.md)

Returns a new STimestamp instance set to the current date and time in the
given time zone.

## Parameters

| :--------- | :------- | :------------------------------------------------------------------------------------------------------------------------------ |
